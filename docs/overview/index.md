# 概述

LangChain 是一个 **Agent 工程平台**。Clay、Rippling、Cloudflare、Workday 等公司的 AI 团队都在使用 LangChain 的产品来构建可靠的 Agent。

## 三层产品体系

| 产品 | 定位 | 适合场景 |
|------|------|----------|
| **Deep Agents** | 开箱即用的 Agent Harness | 快速构建复杂 Agent，自带子 Agent、文件系统、长期记忆 |
| **LangChain** | Agent 开发框架 | 需要自定义 Agent 行为，灵活组合模型、工具、链 |
| **LangGraph** | 底层编排运行时 | 复杂工作流，需要持久化、人工介入、时间旅行 |

## 选型建议

- 🆕 **刚入门** → 从 [Deep Agents](/deepagents/) 开始，最快上手
- 🔧 **需要定制** → 用 [LangChain](/langchain/) 的 Agent 模块
- 🏗️ **复杂工作流** → 用 [LangGraph](/langgraph/) 底层编排
