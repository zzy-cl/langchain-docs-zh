# Studio 调试

## 这是什么？

LangSmith Studio = LangGraph 的"可视化调试器"。在浏览器里看到你的图长什么样，每一步的状态是什么。

## 功能

| 功能 | 说明 |
|------|------|
| **图可视化** | 看到节点和边的连接关系 |
| **状态查看** | 每个节点执行后的状态 |
| **时间旅行** | 回溯到任意历史节点 |
| **断点调试** | 在指定节点暂停 |
| **实时流** | 实时看到执行过程 |

## 使用方式

访问 [smith.langchain.com](https://smith.langchain.com)，选择你的项目，进入 Studio。
